# Messenger

Простой и легковесный мессенджер, реализованный на Java/TypeScript.

##  Описание

Этот проект представляет собой веб-приложение для обмена сообщениями между пользователями.

Основные цели проекта:
- реализация обмена сообщениями в реальном времени
- изучение сетевого взаимодействия
- демонстрация архитектуры клиент-сервер

##  Возможности

-  Отправка и получение сообщений
-  Поддержка нескольких пользователей
-  Реальное время
-  Хранение сообщений
-  Базовая авторизация

##  Технологии

- Язык: Java/TypeScript
- Фреймворк: Spring Framework
- База данных: PostgreSQL
